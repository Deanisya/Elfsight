# Elfsight Test Task

## Локальный запуск

1. Установите зависимости:
   ```bash
   npm install
   ```
2. Запустите проект в режиме разработки:
   ```bash
   npm start
   ```
3. Откройте в браузере:
   [http://localhost:3000](http://localhost:3000)

## Дополнительно

- Проверка линтера:
  ```bash
  npm run lint
  ```
- Важно: данные из API могут не загружаться без включенного VPN.
